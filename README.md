# flutter_pip

you must config 
```
<key>UIBackgroundModes</key>
<array>
	<string>audio</string>
	<string>fetch</string>
	<string>location</string>
	<string>processing</string>
</array>
```

to `/ios/Runner/Info.plist` of project.

and add the file `background_video_of_pip.mp4` to Runner folder in ios.
